Descripción detallada

Código - Lógica de control con el servidor Cerebro y gestión del audio recibido.

Utilizando una máquina de estados de tipo Mealy gestiona la conexión con el servidor Cerebro a través de Websocket. Diferencia entre texto o datos binarios recibidos, para gestionarlo como un comando del sistema o como datos que tiene que enviar al descodificador.
