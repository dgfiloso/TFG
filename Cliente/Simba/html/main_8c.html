Descripción detallada

Código - Cliente del servidor Cerebro. Recibe audio y lo envía por SPI para reproducirlo.

Programa para el módulo ESP8266 utilizando el sistema operativo de tiempo real Simba. Se conecta al sistema Cerebro a través de Websocket siempre como receptor. Una vez conectado espera a que se le comunique con un transmisor para recibir el audio que este envía y reproducirlo mandándolo por SPI.
