Descripción detallada

Código - Cliente del sistema Cerebro. Recibe audio y lo envía para reproducirlo.

Programa para el módulo ESP8266 utilizando el sistema operativo de tiempo real FreeRTOS. Mediante el uso de un servidor UDP recibe audio y lo envía a otro módulo para que lo reproduzca.
